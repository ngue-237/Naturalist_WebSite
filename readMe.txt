consigne d'utilisation:
1- Le projet est structuré selon l'architecture MVC
2- Le dossier sert à faire tout les traitement nécessaire pour les CRUD
3- Le dossier Models permet de faire la déclaration de la classe et des getters et setters
4- Le dossier c'est pour tout ce qui à trait à l'utilisateur
5- lorsque vous souheterai faire un traitement dans un document (models et controlleurs), bien vouloir le faire dans le dossier relatif à votre module (créer un dossier pour votre module)
